# Taller Concurrencia y Paralelismo en Java

Autor: Deiby Martínez

## Descripción
Este repositorio contiene el desarrollo del taller sobre concurrencia y paralelismo utilizando Java.

## Ejercicio 1
Programa que calcula la suma de los números del 1 al 1,000,000 utilizando 4 hilos en paralelo.

## Ejercicio 2
Simulación de una cuenta bancaria compartida donde varios clientes intentan retirar dinero al mismo tiempo.

## Archivos del proyecto
- SumaHilo.java
- ParalelismoSuma.java
- SimulacionBanco.java
